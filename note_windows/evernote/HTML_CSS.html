CSS总结
CSS :层叠样式表   Cascading Style Sheet
                作用：用于美化页面
        2.怎么写?
                有三种写法
                1.行内样式:  style="属性名:值;属性1：值1;"  style是一个属性
                2.内嵌样式
                        <style type="text/css">
                                样式选择器{
                                        属性名:值;
                                }
                        </style>
        3.外部样式
                Link引入(推荐)
                        <link rel="stylesheet" type="text/css" href="样式名件名" />
                import导入
                        <style>
                                @import url('01.css');
                          </style>
                4.优先级问题:
                        就近原则:(内部>外部,下部>上部)
        div:具有换行功能，而span没有这个功能
        选择器:
                html标签选择器
                class类选择器,定义时用.cc,引用时用class="cc"
                id选择器   定义时用#div1,引用时用id="div1"
        L-V-H-A   在visited中设置font-size不起作用
        定位:
        position: absolute绝对定位，它是相对于浏览器左上角(0,0)坐标而言，并且后面的内容会往前跑
                  relative:相对定位，它是相对于自己原来的位置而言，并且后面的内容不会有任何变化
        float:left,right
        clear:清除左右的浮动
        display:设置可见性,none不可见，block可见
