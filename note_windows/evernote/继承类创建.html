XML方式下，创建继承类的对象：

student 继承了Person ，要在配置文件中，对于子类的 bean ，拥有父类的属性，应该增加一个属性： 

代码演示如下：
父类代码：
public class Person {
     private String name;

     public String getName() {
           return name ;
     }

     public void setName(String name) {
           this.name = name;
     }
     
     public void outName(){
          System. out.println(this .name );
     }
}

子类代码：
public class Student extends Person{
     public void say(){
           this.outName();
     }
}

这样，就可以直接通过容器获得子类对象，并且，继承父类的方法。
