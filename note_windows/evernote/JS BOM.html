BOM:
        Browser Object Model  浏览器对象模型
            Window:
                Window 对象是 JS层级中的顶层对象,代表一个浏览器窗口或者一个框架,会在 <body> 或 <frameset> 每次出现时被自动创建.
                Window.open()的第三个参数十分有用,要重点关注
                父子窗口传数据(子-》父:parent,  父----子：frames[0])
                打开与被打开窗口之间传数据:opener
                打开模式对话框:使用window. showModalDialog 方法实现
            Location:
                对象实际上是JS对象，而不是HTML DOM对象,包含有关当前 URL 的信息。
                herf属性
            History:由一系列的URL组成。这些URL是用户在一个浏览器窗口内已访问的URL,也就是历史记录.
                    go(),back(),forward()
            Document:代表当前的页面.
                     document.getElementById("id");
                     document.getElementsByName("name");//返回数组
            Navigator:包含当前浏览器的信息;
                      navigator.appName,appVersion,platform
            Screen:包含当前屏幕信息
                   width,height
     常用事件
          弹框
          1.alert();警告
          2.confirm();弹出一个确定取消框,点确定返回true,点取消返回false
          3.prompt();弹出一个提示输入框
        漂浮广告的思路:
                1,得到浮动块;通过document.getElementById("id属性值");
                2,为浮动块重新设置坐标,
                        通过得到的浮动块得到style属性,用随机数为style设值
                3,设置定时器
                        通过setTimeout();
        history:
                history.go(1),前进,相当于history.forward()
                history.go(-1)，后退,相当于history.back();
        Location:代表的地址栏,herf是它的一个属性
        mouse:鼠标
                onmouseover:鼠标移进
                onmousemove:鼠标在上面
                onmouseout:鼠标移出
                onclick:鼠标点击
        onload/onuload:加载/卸载
          window.onload = loadss;//不加括号,加载时调用的函数
          window.onunload = unloadss;//卸载时调用的函数(联天室非正常退出可以用到);
        聚焦/离焦 
          onfocus:得到焦点
          onblur:失去焦点
        键盘事件
          onkeydown:按下键盘
          onkeypress:压住键盘
          onkeyup:弹起键盘
        提交/重置事件
                submit/reset
                1.是写在form表单中，而不是写在提交按钮中
                2.调用格式:onsubmit="return 函数名()"这里的return是验证是否提交成功
                3.被调用的函数中一定要返回布尔值
        选择与改变事件
                select/change
                可以实现级联菜单
                new Option("文本","值(是发送给服务器的)");
