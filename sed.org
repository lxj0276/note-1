* 简单应用
** hive中替换None为0
要先确定哪些字段需要被替换,然后前边加上if(
sed "s/\bif(\([^ |^,]*\)/if(\1 = \'None\', 0, \1)/" aaa.txt
sed "s/\bif(\([^ |^,]*\),2)/if(\1,2) = null, 0, \1,2))/"

** 把最后一个字符╪替换为空
sed -e 's/\(.*\)╪/\1/' test
sed -i 's/\(.*\)╪/\1/' test

** 替换╪为制表符号
sed -i "s/╪/\\t/g" src_goods_channel_relation_his2015-04-28
