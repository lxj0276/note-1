* Knowledge
  - Add directory to IPFS
  - Access files with gateway.ipfs.io
  - Get an IPNS ID or peer ID
  - Modify you DNS records for an IPFS site


** Add dir
  ipfs add -r .
** Publish
  ipfs name publish <hashCode>
** Change your DNS TXT
  dnslink=/ipns/<peer id>
